## 2. LDA_분석_웹_App
####app.R
- 사용자 입력한 파일에 대해 LDA 기반 동향 분석 One-Stop 서비스
- 텍스트(문서)들의 집합에 포함된 주제를 분석하고, 이를 토대로 기간별 동향을 파악하는 서비스
- R Shiny로 Webapp 구현


## 3_키워드_네트워크_웹App
